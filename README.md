# pyrate
## About Me
Calculate HONO reaction rates and plot photostationary state

## Installation
```
pip install pyrate
```
## Usage
Uses the output of the F0AM v4.2.2 code written by Bode Hoover. Reaction rates and names are determined by the MATLAB script.
